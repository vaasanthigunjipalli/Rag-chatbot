# Rag-chatbot
## Description
A local question-answering chatbot using LangChain. It ingests a text corpus, indexes it with embeddings (SentenceTransformers), and answers user queries by retrieving relevant passages and generating answers with an LLM.

## Features
- **RAG Pipeline:** Document ingestion, embedding (MiniLM), FAISS/Chroma vector store.
- **LangChain Agent:** Custom agent with retrieval tool and LLM.
- **Local Models:** Uses open-source models (no cloud APIs).
- **Testing:** Includes tests for retrieval accuracy and answer validity.
- **Skill Highlights:** Python, NLP, embeddings, vector search, prompt engineering, LangChain agents.
